# Backend 開發紀錄 - 岩壁計算機 (SA-HAHAHA)

## 📌 目前進度摘要
後端「資管系畢業初審大腦」已完成核心架構實作。具備自動爬取、系所判斷、細項學分分類（全人教育、必修、選修、通識）與擋修預警功能。

---

## ✅ 已實作功能 (Implemented)

### 1. 基礎架構
- [x] **CORS 設定**: `main.py` 已加入 `CORSMiddleware`。
- [x] **Health Check**: 新增 `/health` 接口。
- [x] **測試儀表板**: 新增 `index.html` 供快速開發測試 (http://localhost:8000/)，並支援四個主要的進度條（全人教育、必修、選修、通識）。

### 2. 爬蟲服務 (`FjuScraperService`)
- [x] **API 轉向**: 成功轉向直接對接輔大 SIS 內部 API (JSON 格式)。
- [x] **JWT 登入**: 實作 `LdapLogin` 取得 JWT Token 並傳入後續請求。
- [x] **SSL 豁免**: 處理學校系統憑證過期問題。
- [x] **系所判斷**: 內建 58 個系所代碼對照表，從學號自動判斷。

### 3. 審查大腦 (`Auditor Logic`)
- [x] **細項學分分類**: 簡化並精確化分類邏輯，分為必修、選修、全人教育。
- [x] **全人教育整合**: 
    - 體育類課程（如羽球、桌球等）自動歸類至「核心課程」。
    - 外國語文類課程（如英文、法文、日文等）自動歸類至「基本能力」。
- [x] **通識分類**: 自動判斷人文、自然、社會科學領域。
- [x] **預警系統**: 自動檢查擋修關係（如系分與專題）。

---

## 📄 學分分類架構與關鍵字對照表

| 分類 | 門檻 (學分) | 關鍵課程關鍵字 |
| :--- | :---: | :--- |
| **必修** | 64 | 排除全人教育後的必修課程 |
| **選修** | 32 | 任何系選修或自由選修 |
| **全人核心** | 8 | 大學入門、人生哲學、專業倫理、企業倫理 + (體育/羽球/桌球/游泳/網球/排球/籃球) |
| **基本能力** | 12 | 國文 + (外語/外國語文/英語/英文/法文/德文/日文/西班牙文/韓文) |
| **通識** | 12 | 人文與藝術、自然與科技、社會科學 |
